# Tutorial Round - "Simulator Practice"

Source basis: `docs/prosperity_wiki_raw/11_tutorial_round.md`.

## Objective

Write a first Python program to trade `EMERALDS` and `TOMATOES`, upload it to the trading simulator, test its performance, and become familiar with the Graphical User Interface and core controls of Prosperity 4.

## Tradable products

| Product name | Symbol |
| --- | --- |
| Emeralds | `EMERALDS` |
| Tomatoes | `TOMATOES` |

## Product behavior hints

The source states:

- `EMERALDS` are precious gemstones with quite a stable value.
- `TOMATOES` tend to fluctuate over time.

## Algorithmic challenge details

- Challenge name: "Simulator Practice".
- The simulator is available during the journey to Intara.
- Trading options in the simulator are limited.
- Players can test Python programs as many times as they like.
- Feedback is almost instant.
