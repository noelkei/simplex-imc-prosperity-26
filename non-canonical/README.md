# Non-Canonical Drafts

This folder is informal personal scratch space. It is outside the formal round workflow.

Use it for:

- rough notes
- temporary experiments
- draft code
- local ideas
- personal work in progress

Do not use it as:

- a source of Prosperity facts
- evidence for strategy selection
- a reference implementation
- a substitute for round-local specs, EDA summaries, run summaries, or debug notes
- a final submission location

Agents should ignore this folder unless the user explicitly points to a specific file or asks to use a personal draft. Even then, treat the content as non-authoritative context and move any useful conclusion into the formal round workflow before relying on it.

Formal round work belongs under `rounds/round_X/`. Official knowledge belongs under `docs/`.
