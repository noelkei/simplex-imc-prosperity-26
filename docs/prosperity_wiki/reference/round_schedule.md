# Round Schedule

Source basis: `docs/prosperity_wiki_raw/04_round_schedule.md`.

The source states:

- The schedule is subject to change based on unforeseen occurrences.
- New round opening is indicated by email and Prosperity Discord message.
- If anything takes longer than expected, a delay notification is sent.
- Hours required for scoring are considered part of the following round's total time.
- Timezone is always CEST (UTC+2:00).
